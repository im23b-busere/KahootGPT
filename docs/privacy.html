Privacy Policy

Data Collection & Use
QuizGPT does not collect, track, or transmit any personal information. All configuration (such as the OpenAI API key, model selection, and click preferences) is stored locally in your browser using Chrome's storage.local API.

Third-party Services
QuizGPT does not use any external analytics or data tracking. If you enable optional payments in the future (e.g., via Stripe or ExtensionPay), their respective privacy policies will apply. At the moment, QuizGPT does not integrate any external payment system.

Data Storage
Your API key and preferences remain on your local device. Nothing is sent to any backend server operated by us.

Changes to This Policy
We may update this policy. Changes will be reflected here and dated accordingly.
