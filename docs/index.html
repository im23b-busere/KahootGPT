QuizGPT – Kahoot Auto Answer

QuizGPT is a Chrome Extension that automatically answers Kahoot questions using OpenAI (GPT-4o, GPT-3.5, etc.).

  • ⚡ Live Kahoot detection
  • 🤖 AI-generated answers
  • 🎯 Optional highlight + auto-click
  • 🔐 No data collection – 100% local
